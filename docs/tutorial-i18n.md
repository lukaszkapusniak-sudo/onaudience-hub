# `tutorial-i18n.js` — tutorial strings

**Path:** [`www/hub/tutorial-i18n.js`](../www/hub/tutorial-i18n.js)

Holds **localized metadata** (`LANG_META`) and **per-step content** (`STEP_I18N`) for the interactive tutorial consumed by [`tutorial.md`](tutorial.md).

Edit this file to change tutorial wording or add languages; `tutorial.js` selects strings based on the active language key.
